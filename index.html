CoexistAI

View on GitHub →

CoexistAI is a modular, developer-friendly research assistant framework that enables you to build, search, summarize, and automate research workflows using LLMs. The framework provides a comprehensive suite of tools for modern AI-powered research.

Key Features:

  • Web Exploration: Advanced web scraping and content analysis
  • Reddit Integration: Extract insights from community discussions
  • YouTube Transcript Analysis: Process and analyze video content
  • GitHub Code Exploration: Navigate and understand codebases
  • Map Generation: Create visual knowledge maps
  • Text-to-Podcast: Convert research into audio format
  • MCP Support: Full Model Context Protocol integration for seamless AI agent interaction
